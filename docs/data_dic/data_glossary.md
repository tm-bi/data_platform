
# Dicionário do padrão das abreviações
Este documento tem como objetivo orientar e padronizar as abreviações nas estruturas das bases de dados, normalmente tratada na camada Silver Transacional, isto é, Data de Utilização será dt_utilizacao, por exemplo.

## Tabela das abreviações

| Abreviação | Nome                | Descrição                                                |
|------------|---------------------|----------------------------------------------------------| 
| qtd        | quantidade          | quantidade de itens                                      |
| cod        | código              | numeral ou alphanumérico de um produto, por exemplo      |
| dt         | data                | data (dd/MM/yyyy ou yyyy-MM-dd)                          |
| id         | identificador       | id_venda - número ou código único de uma venda           |
| pgto       | pagamento           | forma de pagamento ou data de pagamento                  |
| nsu        | nº sequencial único | usado para identificar transações com cartões de crédito |
| hr         | hora                | hora (hh:mm:ss)                                          |
| vlr        | valor               | valor monetário ou numérico                              |
